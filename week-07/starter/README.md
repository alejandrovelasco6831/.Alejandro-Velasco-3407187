# Week 7 – Librería de Funciones

## Descripción

En esta actividad se desarrolló una librería de funciones en JavaScript aplicada al dominio Diseño de Interiores, con el objetivo de organizar y reutilizar lógica mediante funciones para procesar información de diferentes proyectos.

## Objetivo

Aplicar los conceptos de:

* Funciones y arrow functions
* Funciones puras
* Validación de datos
* Parámetros por defecto
* Uso de funciones dentro de un flujo de programa
* Template literals

## Funcionalidades

Se implementó un sistema que permite listar proyectos de diseño de interiores, validar cuáles están activos, calcular el valor total de los proyectos y mostrar la información de forma organizada utilizando funciones reutilizables.

starter/script.js

## Cómo ejecutarlo

1. Abrir la terminal en la carpeta principal del proyecto.
2. Verificar que Node.js esté instalado con el comando: node -v
3. Ejecutar el siguiente comando: node week-07/starter/script.js
4. El programa mostrará en consola el reporte de los proyectos.
