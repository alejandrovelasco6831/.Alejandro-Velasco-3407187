# Week 5 – Clasificador del Dominio

## Descripción

En esta actividad se desarrolló un clasificador en JavaScript que permite evaluar un elemento del dominio Diseño de Interiores, utilizando estructuras condicionales para determinar su estado, tipo y nivel de avance.

## Objetivo

Aplicar los conceptos de condicionales (if, else if, else), operador ternario (? :), estructura switch, operador de fusión nula (??), encadenamiento opcional (?.) y uso de template literals.

## Funcionalidades

El sistema permite clasificar proyectos según su porcentaje de avance, validar su estado (activo o inactivo), identificar el tipo de diseño y manejar valores nulos o indefinidos de forma segura.

## Cómo ejecutarlo

1. Abrir la terminal en la carpeta principal del proyecto.
2. Verificar que Node.js esté instalado con el comando: node -v
3. Ejecutar el siguiente comando: node week-05/starter/script.js
4. El programa mostrará en consola la ficha de clasificación.
