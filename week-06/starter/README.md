# Week 6 – Reporte con Bucles

## Descripción

En esta actividad se desarrolló un sistema de reporte en consola para el dominio de Diseño de Interiores, utilizando estructuras de repetición para recorrer y analizar una lista de proyectos.

## Objetivo

Aplicar el uso de bucles (for...of y for clásico) para recorrer colecciones de datos, realizar conteos, cálculos estadísticos y generar reportes organizados en consola.

## Funcionalidades

El programa permite listar todos los proyectos, contar elementos por categoría, calcular el total y promedio de precios, identificar el proyecto con mayor y menor valor, y generar un reporte comparando cada elemento con el promedio.

## Cómo ejecutarlo

1. Abrir la terminal en la carpeta del proyecto.
2. Verificar que Node.js esté instalado con el comando: node -v
3. Ejecutar el siguiente comando:
   node week-06/starter/script.js
4. El programa mostrará el reporte en consola.
