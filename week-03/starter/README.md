# Week 3 – Calculadora del Dominio

## Descripción
En esta actividad se desarrolló una calculadora en JavaScript que permite realizar operaciones relacionadas con el dominio **Diseño de Interiores**, como el cálculo de costos, descuentos y presupuesto.

## Objetivo
Aplicar los conceptos de:

- Operaciones aritméticas (+, -, *, /)
- Asignación compuesta (+=, *=)
- Comparación estricta (===)
- Operadores lógicos (&&, ||)
- Uso de constantes
- Números con separadores (_)

## Funcionalidades

- Cálculo del costo total del proyecto
- Costo por metro cuadrado
- Aplicación de descuentos
- Validación de presupuesto
- Evaluación de condiciones para iniciar el proyecto

## Archivo principal

starter/script.js

## Cómo ejecutarlo

Abrir la terminal y ejecutar:

node starter/script.js

## Dominio utilizado

Diseño de Interiores
