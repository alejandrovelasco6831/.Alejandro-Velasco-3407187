# Week 1 – Ficha de Presentación del Dominio

## Descripción
En esta actividad se creó un script en JavaScript que muestra en consola una ficha de presentación del dominio **Diseño de Interiores**.

## Objetivo
Practicar los conceptos básicos de JavaScript utilizando:
- console.log()
- tipos de datos
- valores numéricos
- valores booleanos
- operaciones simples

## Archivo principal
starter/script.js

## Cómo ejecutarlo

Abrir la terminal y ejecutar:

node starter/script.js

## Dominio utilizado
Diseño de Interiores
