# Week 2 – Ficha de Datos del Dominio

## Descripción
En esta actividad se desarrolló una ficha de datos en consola usando variables, tipos de datos y conversiones aplicadas al dominio **Diseño de Interiores**.

## Objetivo
Practicar el uso de:

- Variables con const
- Tipos de datos (string, number, boolean, null)
- typeof
- Conversiones de datos
- console.log() con template literals

## Archivo principal
starter/script.js

## Cómo ejecutarlo

Abrir la terminal y ejecutar:

node starter/script.js

## Dominio utilizado
Diseño de Interiores
