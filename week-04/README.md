# Week 4 – Generador de Mensajes

## Descripción
En esta actividad se desarrolló un generador de mensajes en JavaScript que muestra información de un proyecto del dominio **Diseño de Interiores**, aplicando transformaciones de texto y validaciones.

## Objetivo
Aplicar los conceptos de:

- Métodos de strings (`trim()`, `toUpperCase()`, `toLowerCase()`)
- Extracción de texto (`slice()`)
- Búsqueda en strings (`includes()`, `startsWith()`, `endsWith()`)
- Template literals
- Generación de mensajes dinámicos en consola

## Funcionalidades

- Limpieza y transformación de nombres
- Generación de ficha de datos en consola
- Validación de códigos y descripciones
- Creación de notificaciones dinámicas

## Archivo principal

starter/script.js

## Cómo ejecutarlo

Abrir la terminal y ejecutar:

node starter/script.js

## Dominio utilizado

Diseño de Interiores
